jisho

This is inspired by the jisho.org Japanese-English web dictionary. For more information regarding search options, you can visit their official documentation. They also have a forum page for further discussions.

This is created through Flask, a popular web framework in Python. The Python API jisho-api is also used to retreive information regarding the requested keyword. The latest version of Bootstrap framework was also used as the frontend toolkit.

Visit therepository, and feel free to be a contributor! This site is hosted through PythonAnywhere. If you have inquiries, please email me at marinoalaica@gmail.com.
